# 买水机三代分成平台 — 设计文档

**日期**: 2026-06-22  
**版本**: v1.0

---

## 1. 项目概述

基于现有 SmartCloud 代码库，修改为买水机月租 + 三代佣金分成平台。客户每月支付水机租金，租金触发三代佣金自动分配。会员后台可查看以下3代组织图及佣金明细。

---

## 2. 核心业务规则

### 佣金结构
| 层级 | 关系 | 比例 |
|------|------|------|
| 第1代 | 直推（直接推荐） | 40% of 月租 |
| 第2代 | 间接推荐 | 3% of 月租 |
| 第3代 | 三层推荐 | 2% of 月租 |

### 月租模式
- 客户每月支付水机租金（金额TBD，上线前填入）
- 会员资格与月租绑定：停止付款 = 会员暂停
- 支付方式：信用卡（客户自行与银行申请 EPP 分期）
- 支付网关：Curlec（已有集成）

---

## 3. 页面结构

### 公开页面（无需登录）
- **首页** — 品牌介绍 + 水机产品展示（图片、型号、价格）
- **产品页** — 水机产品图、规格说明
- **代理登录** — 邮箱 + 密码
- **代理注册** — 姓名、邮箱、电话、身份证号、推荐人ID（选填）

### 会员页面（需登录）
- **仪表板** — 月租状态、本月佣金、钱包余额、团队人数
- **组织图** — 只显示自己以下3代（非完整树）
  - 第1代：显示每人姓名、加入日期、月租状态
  - 第2代：显示人数总计 + 列表
  - 第3代：显示人数总计 + 列表
- **佣金记录** — 每笔来源、金额、日期
- **提现申请** — 最低 RM100，最高 RM10,000

### 管理员页面
- **管理员登录** — 独立入口
- **会员管理** — 查看/暂停会员
- **佣金批量处理** — 审批/发放
- **提现处理** — 批准/拒绝

---

## 4. 技术架构

### 沿用现有 SmartCloud 栈（不重写）
- **前端**: `index.html` SPA（Tailwind CSS + Vanilla JS）
- **后端**: Node.js + Express (`server.js`)
- **数据库**: MySQL 8.0（现有 schema 直接复用）
- **支付**: Curlec API（月度订阅）
- **认证**: JWT Token（24h 过期）

### 需要修改的文件
| 文件 | 修改内容 |
|------|----------|
| `index.html` | 品牌名、产品内容改为水机、三语言切换 |
| `server.js` | 无需改逻辑，只改产品 seed 数据 |
| `.env` | 品牌名、月租金额（TBD）、Curlec 密钥 |
| `schema.sql` | 无需改结构，只改 seed 产品数据 |

### 三语言实现
- 顶部语言切换按钮：中文 / English / Bahasa Malaysia
- 实现方式：JS 对象存储翻译字符串，切换时替换 DOM 文本
- 不需要引入 i18n 框架（保持简单）

## 5. 数据流

### 月租支付触发佣金
```
客户付月租 → Curlec Webhook → server.js 更新订单状态
→ 触发 /api/commissions/calculate
→ 向直推代理发放 40%
→ 向第2代代理发放 3%
→ 向第3代代理发放 2%
→ 各代理钱包余额自动更新
```

### 组织图查询
```
GET /api/agents/{agentId}/team-tree
→ 查询 level1（直接推荐，含姓名+状态）
→ 查询 level2（level1 的下线，含人数+列表）
→ 查询 level3（level2 的下线，含人数+列表）
→ 返回结构化 JSON
```
