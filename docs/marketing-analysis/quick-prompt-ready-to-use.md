```

- 目标受众：[填入目标受众描述]
- 用户的表面问题：[填入用户看到的具体问题]
- 问题的根本原因：[填入你揭示的深层原因 - 这是框架化的核心！]

- 品牌标语：[填入你的品牌标语]

【生成任务】

请按照以下结构生成7张营销贴纸的文案。每张的格式和要求我都明确给出了。

【第1张 - 问题陈述】

标题格式："[行为A]，却[结果B]"
副标题格式："是因为[根本原因]"

任务：用3秒吸引注意力，让用户感到"对，就是我"

标题：

② 指出用户的矛盾（"你一边...一边..."）
③ 转向根本原因（"所以..."）
④ 最有力的洞察句（用【】标注）

任务：消除自责，指向心理根源

① 表现层（用户会有什么行为）- 列举3-4个具体例子
② 情绪层（用户会感到什么）
③ 根本原因（为什么会这样 - 深度分析）

标题：
表现层：

【第4张 - 第二种症状】

标题格式："「第二种[问题分类]：[症状名称]」"

内容同第3张（表现→情绪→原因→洞察）
这个症状应该与第3张互补，覆盖不同的用户

任务：扩大覆盖面，多角度诊断

请输出：
标题：
表现层：
情绪层：

【第5-6张 - 深化信息（选一种）】

请根据你的内容选择最合适的一个，保持前面的风格。

---

【第7张 - 品牌承诺与行动号召】

标题格式："「[品牌名]」\n[品牌Slogan]"

内容结构：
① 开头同理："如果你...那不是...而是..."

【特别要求】

生成时请记住这些原则：

✓ 文案风格：
  - 用"你"开头，制造直接对话感
  - 使用具体生活场景，避免抽象概念
  - 每个句子都有节奏感
  - 最重要的洞察放在最后

✓ 心理学原则：
  - 包含同理心（"我理解你"）
  - 包含去责任化（"不是你的错"）
  - 引发自我诊断（让用户自己发现问题）
  - 传递希望感（"可以改变"）

✓ 转化原则：
  - 第1张的吸引力要突出（让人停留）
  - 第2-3张要建立信任（让人认同）
  - 第4-5张要引发思考（让人投入）
  - 第7张要明确指引（让人行动）
```

## ⚡ 快速优化技巧

### 如果生成的洞察不够深入

在Prompt最后添加这句话：

```
特别强调：最后一句的【洞察句】非常关键。
它不应该是"你有焦虑"这样简单的总结。
而应该是触及内心的、有冲击力的洞察。

例如：
✗ 不好："你需要建立自信"
✓ 很好："当你的价值被他人定义，你就永远无法站稳"
✓ 很好："你对成功的恐惧，远大于对失败的恐惧"
✓ 很好："你一直在为不曾拥有过的爱而战斗"

请确保每个【洞察句】都这样有力。
```

### 如果框架化不够新颖

在开头添加这个前置问题：

```
在生成文案之前，我想先确认框架化的创新性。

对于[问题]这个议题，
我看到的新视角是：
"不是[常见看法]，而是[我的新看法]"

具体是：[你的新框架]

你认为这个框架化足够新颖吗？
有没有更深入的角度我可以挖掘？

（只有我满意框架化后，才请生成完整的7张文案）
```

### 如果生成的文案太学术

在特别要求末尾添加：

```
额外提醒：避免学术性语言。
✗ 不要说："心理补偿机制"
✓ 要说："一花钱就放松，买完就后悔"

✗ 不要说："自我评估系统偏差"
✓ 要说："你总觉得自己不够好"

请用日常语言，就像在和朋友聊天。
```
