# 营销系列创建实操模板

## 第一步：选择你的主题

### 选题框架

问自己这几个问题：

```
1. 我的目标受众有什么样的"隐性痛点"？
   ├─ 不是表面问题（如：不会理财）
   └─ 而是深层问题（如：对金钱的心理恐惧）

2. 这个痛点是否能被"重新框架化"？
   ├─ "不是X的错，而是Y的问题"
   └─ 例：不是你不够努力，而是你的自我价值评估有问题

3. 这个问题是否有2-3个清晰的"症状"可以诊断？
   └─ 症状越具体越好（行为+情绪）

4. 你是否有解决方案可以提供？
   └─ 课程、服务、产品、社群等
```

### 主题方向参考

| 领域 | 痛点 | 重新框架 | 典型症状 |
|------|------|--------|--------|
| **自我成长** | 总是坚持不下去 | 不是意志力不够，而是动机系统破裂 | 三分钟热血、自我否定、目标不清 |
| **亲密关系** | 总是遇到渣男 | 不是运气差，而是选择标准有问题 | 忽视红旗、低自我评价、渴望拯救 |
| **职业发展** | 薪资停滞 | 不是能力差，而是自我定位有误 | 不敢要求加薪、自我贬低、假谦虚 |
| **身体健康** | 减肥总反弹 | 不是毅力差，而是根本原因没解决 | 节食→暴食、自我惩罚、情绪化进食 |
| **创意工作** | 创意枯竭 | 不是没想法，而是内心有审查官 | 完美主义瘫痪、恐惧批评、过度思考 |

## 第三步：找到"重新框架化"的角度（最重要！）

这是整个系列的灵魂。

### 框架化模板

```
【普遍认知】
"我的问题是：_________"
例：我赚不到钱

【表面原因】（人们通常这样解释）
"因为我：_________"
例：因为我不够努力、不懂理财、没机遇

【你的独特视角】（重新框架化）
"但实际上，问题不是_________"
"而是_________"
例：问题不是赚多少，而是你对钱的心理关系破裂了

【这个视角为什么对？】
- 原因1：_________
- 原因2：_________
- 原因3：_________
```

### 好的框架化例子

```
✓ 不是"你不漂亮"，而是"你的内在没有被看见"
  → 这转移了焦点从外部→内部，从不可改变→可改变

✓ 不是"你没天赋"，而是"你的自信系统被破坏了"
  → 这给了人希望，因为系统可以重建

✓ 不是"他不爱你"，而是"你的界限意识不清"
  → 这转移了被动受害→主动承担

✗ 不好的例子："不是你的错，就是我的错"
  → 这没有指向任何可改变的点
```

## 第四步：设计2-3个诊断症状

### 症状设计框架

每个症状应该有这个结构：

```
【症状名称】（给问题起一个名字）

【表现层面】（外在可见的行为）
- 你会_________
- 你常常_________
- 你总是_________

【情绪层面】（内在的感受）
- 你感到_________
- 你害怕_________
- 你渴望_________

【根本原因】（为什么会这样）
└─ 这反映出：_________
   因为你的_________信念是：_________
```

### 例子

```
【症状：补偿式消费】

【表现层面】
- 你在压力大时就想买东西
- 你经常在情绪低落后才发现自己又买了什么
- 你购物清单里很多从没用过的东西

【情绪层面】
- 买的瞬间感到放松
- 事后感到内疚和失控
- 你在逃避某种感受

【根本原因】
└─ 这反映出：你用物质代替了情绪处理
   因为你的内在信念是："我的情绪不值得被理解"
```

### 设计检查清单

对每个症状问自己：

- [ ] 这个症状是否容易被识别？（用户看到会说"对，就是我"）
- [ ] 症状描述是否具体而非抽象？
- [ ] 是否涵盖了行为+情绪+原因？
- [ ] 末尾的"洞察"是否足够深入？
- [ ] 不同症状之间是否有差异？（不要重复）

## 第五步：创建7张贴纸文案

### 贴纸1：问题陈述（最关键）

```
【标题】- 必须是行为矛盾或反差
格式："你[行为A]，却[结果B]"

例子：
✓ "你存不到钱"
✓ "你总是赚得到，却留不住"
✓ "你很聪明，却找不到自己"
✓ "你在假装不在乎，却在暗夜哭泣"

【副标题】- 解释这个矛盾
格式："是因为[深层原因]"

例子：
✓ "是因为金钱关系没有调整好"
✓ "是因为你的内在评价系统破裂了"
✓ "是因为你从未被看见"

【视觉指导】
- 用最有冲击力的图片
- 模特表情应该是深思、内敛、有故事感
- 字体要大、要有呼吸感
```

### 贴纸2：视角转变（消除自责）

```
【标题格式】："你不是[常见自责]"

例子：
✓ "你不是赚得不够"
✓ "你不是不够漂亮"
✓ "你不是没有能力"

【主文案结构】
段落1：验证他们的感受
"很多人一直想学[技巧]，可都没有发现，
自己和[问题]的关系一直是[情感词]的。"

段落2：转向
"你可能一边想[目标]，
一边又[相反的行为]。"

段落3：关键转向
"于是[结果]。"

段落4：核心洞察
【用方框强调】
"你[现象]，不只是[表面原因]，
而是[深层原因]。"

【例子】
---
你不是赚得不够

很多人一直想学怎么买更多，
却没有发现，自己和金钱的关系
一直是紧张的。

你可能一边想要钱，
一边又害怕有钱后的压力、责任、失去。

于是钱进来了，也很容易流走。

【你留不住钱，不只是能力问题，
而是内在关系问题。】
---
```

### 贴纸3和4：症状诊断

```
【标题格式】："「第[X]种[问题类别]：[症状名称]」"

例子：
✓ "「第一种关系：匮乏感」"
✓ "「第二种模式：完美主义瘫痪」"
✓ "「第一种信念：不配得感」"

【结构】
段落1：症状的表现（列举3-4个具体行为）
- 你总是_________
- 你常常_________
- 你害怕_________
- 你渴望_________

段落2：深层机制（解释为什么会这样）
你的内在还活在「[内在模式]」的模式里。

段落3：结果（这个症状带来什么）
因为[原因]，所以[结果]。

【关键洞察】（用醒目背景）
【这个症状的本质是：[一句话洞察]。】

【例子 - 匮乏感】
---
「第一种关系：匮乏感」

你总觉得不够。钱一花就焦虑，
钱一少就恐惧，钱一多也不敢真正享受。
因为你的内在还活在
「随时会失去」的模式里。

当这种心态没有被调整，
花钱就很难宽心留下。

【当情绪没有被看见，
花钱就会变成止痛药。】
---

【例子 - 补偿式消费】
---
「第二种关系：补偿式消费」

有些人花钱，不是真的需要，
而是在补情绪。
压力太了想买，委屈了想买，空虚了想买。
那一刻你买的不是东西，
而是短暂的安慰。

当情绪没有被看见，花钱就会变成止痛药。

【这个症状的本质是：
用物质代替了情绪处理。】
---
```

### 贴纸5-6：信息深化（可选）

```
可选内容方向：
1. 再加一个症状
2. 症状的长期后果
3. "为什么你一直没有改变"的分析
4. 行业/统计数据（如适用）
5. 更多细节的问题探讨

【格式保持一致】
- 标题明确
- 信息分层
- 字数不要超过正文
- 保持色彩和排版一致
```

### 贴纸7：品牌+解决方案（行动号召）

```
【标题】：你的品牌名 + 承诺

格式："「[品牌名]」
[中文标语]"

例子：
✓ "「Master NL」
完整自己·金钱成长"

【主文案】
开头：同理
"如果你总是[症状]，
或一直对[问题]有[情绪]，
那不是你不行，"

转折：原因指向
"而是你的[系统]需要被整理。"

【行动号召框】
格式：
"关注我 | [具体承诺]，[具体价值]"

例子：
✓ "关注我 | 一起看懂你的金钱模式，
找回更稳定的丰盛"
✓ "加入我 | 用4周时间重建你的自信系统"
✓ "关注我 | 拆解你的选择标准，
找到适合你的关系模式"

【下方补充】
通常放4个小icon，代表课程/服务的4个支柱
例如：
💝 金钱关系 | 🔑 匮乏模式 | 💳 情绪清费 | 👤 自我价值

【品牌方信息】
logo + 品牌全称 + 标语
```

## 第六步：视觉设计指导

### 色彩系统

```
选择你的配色：

推荐配置1（温暖高级）：
背景：米色/奶油白 + 象牙色
强调：酒红/棕红
字体：深棕/黑色

推荐配置2（现代简约）：
背景：雾白/清水白
强调：深蓝/深灰
字体：深灰/黑色

推荐配置3（温柔知性）：
背景：米白/杏色
强调：莓红/深紫
字体：深紫/黑色

【记住】：
- 不要超过3种主色
- 强调色必须够深
- 背景必须是柔和的
```

### 排版系统

```
【标题】
字体：黑体或现代无衬线体
大小：相对最大
颜色：深色（黑/深棕/深蓝）
行距：1.2倍
对齐：居中或左对齐

【副标题】
字体：正文字体稍粗
大小：中等
颜色：中等灰色或强调色
行距：1.5倍

【正文】
字体：宋体或清晰无衬线体
大小：标准（14-16px）
颜色：深灰（#333）或黑色
行距：1.8倍（重要！阅读舒适）

【强调句】
背景：淡色强调色 或 白底深色边框
内间距：适当留白
字体：稍粗或斜体
颜色：深色
```

### 装饰元素

```
类型建议：
✓ 植物（叶子、花、枝条）- 温暖、生机
✓ 几何图形（圆、线条、星星）- 现代、秩序
✓ 手绘感元素 - 温度、人性
✓ 空白 - 高级感、呼吸感

✗ 避免：
- 过度装饰（不要铺满）
- 复杂图案（易显得廉价）
- 多色渐变（显得不专业）
- 贴纸感（显得幼稚）

【平衡】：
7张贴纸中，装饰应该逐步增加
→ 1张简洁
→ 2-6张逐步加入元素
→ 7张最丰富（但仍保持秩序）
```

## 第七步：完整性检查

### 创建前检查清单

- [ ] 主题选择是否聚焦？（不要太宽泛）
- [ ] 目标受众定位是否清晰？
- [ ] "重新框架化"的角度是否新颖？
- [ ] 2-3个症状是否足够具体且互不重复？
- [ ] 7张贴纸的故事弧线是否完整？
  - [ ] 第1张能在3秒内吸引注意力吗？
  - [ ] 第2-3张能建立同理和信任吗？
  - [ ] 第4-5张能让用户自我诊断吗？
  - [ ] 第6-7张的行动号召是否清晰？
- [ ] 整个系列是否有统一的视觉风格？
- [ ] 文案是否避免了行业术语，足够易懂？
- [ ] 是否有"洞察"时刻让人产生"啊！"的感觉？

### 创建后优化

**发布前问自己：**

1. **第一眼吸引力** - 有陌生人会停留看第1张吗？
2. **认同感** - 目标受众会说"对，就是我"吗？
3. **参与度** - 用户是否想继续看下去？
4. **转化力** - 用户是否清楚"下一步要做什么"？
5. **记忆度** - 用户是否会记住你的品牌？
6. **分享度** - 用户是否会想分享给朋友？

## 额外资源：不同领域的示例框架

### 示例1：自我价值主题

```
【主题】：内在价值系统

【问题陈述】："你很努力，却总是觉得自己不够好"

【重新框架】：
不是你能力不够，
而是你的内在评价标准被扭曲了。

【症状1】：冒牌者综合征
- 你总是怀疑自己
- 成就总被你归功于运气
- 你害怕被人发现"真实的你"
└─ 根本原因：你从未相信过自己值得成功

【症状2】：讨好型人格
- 你总是优先考虑别人的需要
- 你很难说"不"
- 你需要所有人的认可才能安心
└─ 根本原因：你的价值建立在别人的评价上

【解决方案】："重建你的自我评价系统"
```

### 示例2：创意工作主题

```
【主题】：创意枯竭和内心审查官

【问题陈述】："你有很多想法，却总是完成不了"

【重新框架】：
不是你没创意，
而是你内心的"审查官"把你的创意掐死了。

【症状1】：完美主义瘫痪
- 你总是觉得"还不够好"
- 你无法交付任何作品
- 你恐惧别人的评价
└─ 根本原因：你设定的标准远高于现实

【症状2】：创意恐惧
- 你害怕创意被否定
- 你害怕被看不起
- 你需要他人的确认才能前进
└─ 根本原因：你把自我价值和作品绑定了

【解决方案】："释放你的创意，允许它不完美"
```

### 示例3：亲密关系主题

```
【主题】：选择标准和边界

【问题陈述】："你总是遇到渣男，或找不到合适的人"

【重新框架】：
不是你运气差，
而是你的选择标准（或缺乏标准）被内心恐惧主导了。

【症状1】：忽视红旗
- 你总是看不到对方的缺点
- 你告诉自己"他会改变"
- 你甚至为对方找借口
└─ 根本原因：你渴望拯救，而非被爱

【症状2】：低自我评价
- 你觉得自己不值得被好好对待
- 你接受委屈的关系
- 你把伤害当成爱的证明
└─ 根本原因：你的价值感来自被需要，而非自我肯定

【解决方案】："建立你的关系底线和自我价值"
```

## 总结：快速创建清单

```
1. 【主题选择】5分钟
   ✓ 选择一个痛点
   ✓ 确认有解决方案

2. 【目标受众】10分钟
   ✓ 填写用户画像表

3. 【重新框架】20分钟
   ✓ 找到独特的视角
   ✓ 这是灵魂，花时间

4. 【设计症状】30分钟
   ✓ 2-3个具体症状
   ✓ 每个包含行为+情绪+原因

5. 【写文案】60-90分钟
   ✓ 先写粗稿
   ✓ 再逐张优化

6. 【设计视觉】120-180分钟
   ✓ 选择配色系统
   ✓ 创建排版规范
   ✓ 设计7张贴纸

7. 【完整性检查】20分钟
   ✓ 走过检查清单
   ✓ 做最后调整

**总耗时：4-6小时可以完成一个完整系列**
```
