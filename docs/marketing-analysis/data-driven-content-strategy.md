## 第七部分：数据呈现方式

### 📊 文案中的数据呈现

**不好的方式：**
```
"我们的课程很有效"
"很多人都改变了"
"效果显著"
```

**好的方式：**
```
"我们的课程改善率达89%"
"500+学员参与，92%推荐"
"平均焦虑指数从8.5降到2.1（75%改善）"
```

### 📊 配图中的数据呈现

建议：
- ✅ 使用数据图表（柱状图、曲线图）
- ✅ 对比视觉化呈现
- ✅ 百分比的明显标注
- ✅ 在Master NL风格基础上加数据元素

例子：
```
【改变前】焦虑指数 8.5
【改变后】焦虑指数 2.1
【改善】75% ↓

用对比图表呈现，视觉冲击力强
```

## 第八部分：引用数据的注意事项

### ✅ 正确使用数据

```
✓ 标注数据来源
  "根据美国心理协会2023年研究..."
  
✓ 明确统计范围
  "在XXX人的调研中..."
  
✓ 准确传达意思
  "改善率达89%"不是"100%的人改善"
  
✓ 对标同行
  "相比传统理财课的12%坚持率，我们是79%"
```

### ❌ 避免的做法

```
✗ 制造数据
  不要编造虚假的统计数据
  
✗ 断章取义
  不要扭曲研究的原意
  
✗ 夸大宣传
  "100%的人完全改变"很容易被质疑
  
✗ 无根据宣称
  "科学证明"要标注具体研究
```
