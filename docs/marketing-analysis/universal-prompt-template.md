## 完整万能Prompt

```
我需要你基于《Master NL》金钱心理成长营销系列的框架和策略，
为我创建一个类似的7张营销贴纸系列。

【我的基本信息】
- 品牌/课程名称：[填入你的品牌名]
- 目标受众：[年龄、性别、职业、收入水平]
- 核心痛点（表面现象）：[用户看到的具体问题]
- 根本原因（你要揭示的）：[问题的深层根源]
- 解决方案：[你提供的课程/服务/社群]
- 品牌Slogan：[你的品牌标语]

【创建要求】

请严格按照以下框架创建7张营销贴纸的文案：

【第1张 - 问题陈述】
作用：3秒内吸引注意力，让用户觉得"这就是我"

要求：
- 标题：用"[行为A]，却[结果B]"的格式
- 副标题：解释这个矛盾的原因
- 任务：制造张力，吸引点击

输出格式：
标题：[标题]
副标题：[副标题]
设计建议：[视觉建议]

---

【第2张 - 视角转变】
作用：消除自责，指向心理根源，建立信任

要求：
- 标题格式："你不是[常见自责]"
- 结构：验证感受 → 指出矛盾 → 转向心理 → 核心洞察
- 关键：用"不是...而是..."的转向
- 任务：让用户从自责转向好奇

输出格式：
标题：[标题]
正文第一段（验证）：[内容]
正文第二段（指出矛盾）：[内容]
正文第三段（转向）：[内容]
【核心洞察】（用方框强调）：[最有力的一句话]

---

【第3张 - 第一种症状】
作用：深化问题诊断，让用户自我识别，建立权威性

要求：
- 标题："「第一种[问题分类]：[症状名称]」"
- 结构必须包含三层：
  ① 表现层（外在行为）
  ② 情绪层（内在感受）
  ③ 根本原因（深层机制）
- 结尾：一句有力的洞察（最重要！）

输出格式：
标题：[标题]
表现层：[列举3-4个具体行为]
情绪层：[用户的内在感受]
根本原因：[为什么会这样]
【洞察句】（用醒目格式）：[一句话总结这个症状的本质]

---

【第4张 - 第二种症状】
作用：扩大覆盖面，多角度诊断，深化认同

要求：
- 标题："「第二种[问题分类]：[症状名称]」"
- 结构同第3张（表现→情绪→原因→洞察）
- 这个症状应该与第3张不同但互补

输出格式：
（同第3张格式）

---

【第5-6张 - 信息深化（可选，选一种）】
作用：进一步说服、建立紧迫感或更多洞察

选项A：再加一个症状
选项B：症状的长期后果（不改变会怎样）
选项C：行业数据支持（如适用）

请选择对你最有说服力的内容。

输出格式：
[保持前面贴纸的风格和结构]

---

【第7张 - 品牌承诺 + 行动号召】
作用：转化，明确下一步

要求：
- 标题：「[品牌名]」+ [Slogan]
- 开头：同理（如果你...那不是...而是...）
- 中间：承诺（完成这个课程后会怎样）
- 结尾：行动号召框

行动号召格式：
"关注我 | [具体承诺]，[具体价值]"

补充：4个小icon代表你的课程/服务的4个支柱

输出格式：
标题：[标题]
开头（同理）：[内容]
中间（承诺）：[内容]
【行动号召】：关注我 | [承诺]，[价值]
四个支柱：[支柱1] | [支柱2] | [支柱3] | [支柱4]

---

【创建的文案要求】

1. 文案风格要求：
   ✓ 用"你"开头，制造直接对话感
   ✓ 用具体行为描述而非抽象概念
   ✓ 避免行业术语，确保易理解
   ✓ 每个句子都有节奏感
   ✓ 最有力的洞察放在最后

2. 心理学要求：
   ✓ 包含"同理心"（我理解你）
   ✓ 包含"去责任化"（不是你的错）
   ✓ 包含"自我诊断"（让用户自己发现）
   ✓ 包含"希望感"（可以改变）

3. 营销转化要求：
   ✓ 第1张的吸引力不低于5分（满分10分）
   ✓ 第2-3张的认同度不低于8分
   ✓ 第4-5张的参与度要引发用户思考
   ✓ 第7张的行动号召要清晰明确
   ✓ 整个系列要有完整的故事弧线

4. 独特性要求：
   ✓ 框架化必须新颖（不能是常见的说法）
   ✓ 症状必须具体（不能太抽象）
   ✓ 洞察必须足够深（不能浅尝辄止）

【输出格式】

请按照我给的格式输出每一张的文案。
每张贴纸之间用"---"分隔。
在最后给出：
1. 整个系列的故事弧线总结
2. 这个框架化的核心创新点是什么
3. 这套文案为什么对你的目标受众有效

---

【参考框架】

请参考以下心理学原理创作：

框架化策略：
- 不是"[表面原因]"，而是"[深层原因]"
- 这能帮助用户从自责转向好奇和希望

症状诊断策略：
- 每个症状都要包含行为层、情绪层、原因层
- 最后一句话要是最有洞察力的

故事弧线：
问题识别 → 去责任化 → 症状诊断1 → 症状诊断2 → 
信息深化 → 承诺展示 → 行动号召

转化漏斗：
停留（第1张）→ 理解（2-3张）→ 诊断（4-5张）→ 
承诺（6-7张）→ 行动

---

开始生成吧！我已经提供了所有的框架和要求。
请为我输出完整的7张营销贴纸文案系列。
```

## 优化技巧

### 如果AI生成的不够有力

在Prompt中添加这些补充指令：

```
补充指令1：加强洞察力
"请确保第3、4张的【洞察句】都足够深入和有冲击力。
比起'你有焦虑感'，应该是'你的焦虑来自于对被看见的渴望'。
这样的洞察。"

补充指令2：加强具体性
"不要使用抽象概念。例如，不要说'你有心理障碍'，
而要说'你一花钱就会想到妈妈的批评'这样的具体场景。"

补充指令3：加强文案节奏
"请确保文案有节奏感。可以用短句+长句的组合。
重要的洞察用单独一句话表达。"
```

### 如果框架化不够新颖

在Prompt前添加这个前置问题：

```
在创建营销文案之前，请先帮我思考：

对于【[我的主题]】这个问题，
人们通常怎么理解它？
├─ 常见解释1：___
├─ 常见解释2：___
└─ 常见解释3：___

我想要的新框架是：
"不是[常见解释]，而是[我的新视角]"

这个新视角的独特之处：___

你同意这是一个创新的框架化吗？
为什么或为什么不？

（只有我确认框架化足够新颖后，才生成完整的7张文案）
```

## 常见问题解答

### Q: 我应该先填所有信息，还是边填边思考？

**A:** 建议这样做：
1. 先填基本信息（品牌、受众、问题）
2. 停下来思考30分钟：你的独特框架化是什么？
3. 再填根本原因和解决方案
4. 最后发给AI

**为什么？** 框架化是灵魂，值得单独思考。

---

### Q: AI生成的文案需要修改吗？

**A:** 通常需要。建议：

```
✓ 第1张：通常很好，可能只需微调
✓ 第2-3张：检查"不是...而是..."是否够有力
✓ 第4-5张：检查洞察是否足够深
✓ 第7张：检查行动号召是否够清晰

修改建议：
- 把抽象的改成具体的
- 把弱的洞察改成强的洞察
- 把长的句子拆成短句+长句
```

---

### Q: 怎样知道我的框架化是否好？

**A:** 用这个检查清单：

```
✓ 标准1：你的框架化是否指向了可以改变的东西？
  ✗ "你就是这样的人" （不可改变）
  ✓ "你有什么信念让你这样做" （可改变）

✓ 标准2：是否转移了责任？
  ✗ "你太贪心了" （批评）
  ✓ "你的内在模式是什么" （诊断）

✓ 标准3：是否打开了好奇心？
  ✗ "你完了" （绝望）
  ✓ "你想知道为什么吗" （好奇）

✓ 标准4：是否足够新颖？
  ✗ "你需要更努力" （老生常谈）
  ✓ "你的努力本身就是逃避" （新视角）
```

如果都符合，你的框架化就很好。

## 最后的提醒

### ⚠️ 最常见的错误

```
❌ 错误1：框架化太浅
"你不运动是因为你懒"
✓ 改为："你不运动是因为你害怕看到真实的自己"

❌ 错误2：症状太抽象
"你有心理问题"
✓ 改为："一到压力大你就想吃东西，吃完又后悔"

❌ 错误3：洞察不够深
"所以要坚持"
✓ 改为："当你的情绪无人倾听，食物就成了你的知心朋友"

❌ 错误4：文案太长
"长长的一段话..."
✓ 改为：短句 + 逗号 + 短句 + 句号。最后一句最长。
```

### ✅ 最成功的要素

```
✓ 新颖的框架化（用别人没想到的视角）
✓ 具体的症状（用生活场景而非概念）
✓ 深入的洞察（触及内心深处）
✓ 有力的文案（每句话都有节奏）
✓ 清晰的行动（用户知道下一步）
```
